# Customer-Shopping-Behavior-Analytics-Dashboard
data analytics project showcasing customer behavior analysis by using python,sql,powerbi
Retail Customer Analytics using SQL, Python & Power BI
Overview

This project focuses on Customer Shopping Behavior Analysis using SQL, Python, and Power BI to uncover purchasing trends, customer segments, and business insights. The objective is to transform raw customer transaction data into meaningful insights through Data Cleaning, Exploratory Data Analysis (EDA), Data Visualization, and Dashboard Development.

Project Objectives
Analyze customer purchasing behavior and shopping trends.
Perform Data Cleaning and Data Transformation.
Conduct Exploratory Data Analysis (EDA).
Identify customer segments and sales patterns.
Build interactive Power BI dashboards.
Generate actionable business insights for data-driven decision-making.
